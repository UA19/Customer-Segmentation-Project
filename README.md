# Customer Segmentation Project

## Overview

This project uses machine learning (KMeans clustering) to group customers based on their income and spending behavior.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Steps Performed

* Data cleaning and exploration
* Feature selection
* Data scaling
* KMeans clustering
* Elbow method to find optimal clusters

## Results

Customers were grouped into different segments such as:

* High income, high spending (premium customers)
* High income, low spending (target customers)
* Low income, low spending (low priority)

## Business Insights

* Businesses should focus on high spending customers
* Target high income low spending customers with promotions
* Reduce focus on low-value segments
